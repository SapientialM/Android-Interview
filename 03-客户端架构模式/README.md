# 03-客户端架构模式

> Android 开发中架构设计的核心知识。MVVM + 单向数据流 + 依赖注入 + 网络层设计 + 模块化。
> **核心策略**：每个概念都在 Spring 后端有对应物，用类比降低学习曲线。

---

## 学习目标

- [ ] 理解 MVVM 各层职责和数据流向
- [ ] 知道为什么用单向数据流，以及如何实现
- [ ] 能使用 Hilt 做依赖注入（类比 Spring IoC）
- [ ] 能设计带统一错误处理的网络层
- [ ] 了解模块化拆分的基本思路
