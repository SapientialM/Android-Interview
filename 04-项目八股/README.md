# 04-项目八股

> 项目答辩和面试中「介绍你的项目」环节的准备材料。
> 面试官问「讲讲你的项目」时，你需要的所有模板、策略、话术都在这里。

---

## 面试中「项目介绍」环节的本质

面试官让你介绍项目，不是要听你报流水账（做了什么功能），而是要考察：

1. **技术深度**：你理解你用的技术吗？（不是只会调 API）
2. **工程能力**：你考虑过架构、性能、边界情况吗？
3. **沟通能力**：你能把复杂的东西讲清楚吗？

## 准备流程

```

Day 3: 阅读 03-技术难点答辩模板 → 准备 3 个技术亮点故事
```
