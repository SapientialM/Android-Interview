# 06-资源索引

本目录指向仓库中用到的原始资料库和外部推荐资源。如果你想深入了解某个主题，这里是你的进一步阅读清单。

---

## 使用建议

| 你的状态 | 阅读策略 |
|---------|---------|
| **时间紧（冲刺中）** | 只看本仓库的整合文档（[00-15天速成规划/](../00-15天速成规划/) — [05-面试八股/](../05-面试八股/)），不要碰原始资料 |
| **时间宽裕（提前准备）** | 看完整合文档后，按本索引深入原始资料 |
| **面试通过后（持续学习）** | 按 [外部推荐资源](02-外部推荐资源.md) 系统学习 |
