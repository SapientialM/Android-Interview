# 01-Kotlin-速成

> 为 Java 开发者定制的 Kotlin 速成。不教基础编程概念（变量、循环、if-else），只讲 Kotlin 与 Java 的差异。
> 如果时间极紧，直接看 [01-Java转Kotlin速查表](01-Java转Kotlin速查表.md) + [02-协程核心概念](02-协程核心概念.md)。

---

## 阅读策略

| 你的背景 | 推荐阅读顺序 |
|---------|------------|
| **Java 经验丰富** | 01 速查表 → 02 协程 → 03 Flow → 04 陷阱（约 4h） |
| **时间极紧** | 01 速查表 + 02 协程（约 1.5h），其他遇到再看 |
| **需要面试** | 全部阅读，协程和 Flow 是面试必考点（⭐⭐⭐） |

## 学习目标

学完后你应该能：

- [ ] 看懂项目中所有 Kotlin 代码（至少不陌生）
- [ ] 理解协程的挂起和恢复机制（面试必问）
- [ ] 知道 StateFlow 和 SharedFlow 的区别和分别用在什么场景
- [ ] 能写简单的协程 + Flow 代码
